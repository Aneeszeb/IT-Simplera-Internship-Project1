# IT-Simplera Network Administration Internship

## Week 1 — Networking Fundamentals & Network Simulation Tools

### 📋 Project Overview
This project covers the fundamentals of computer networking, including
network devices, IP addressing, and communication models. A simple
LAN topology was designed and implemented using Cisco Packet Tracer
and GNS3.

### 🛠️ Tools Used
- Cisco Packet Tracer
- GNS3
- PuTTY

### 🔧 What Was Implemented

- Configured static IP addressing on all devices
